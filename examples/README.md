# Twelve Data API client for Go examples

This folder contains usage examples for Go.

## Prerequisites

Go 1.23 or later. Use [gvm](https://github.com/moovweb/gvm) for Go version management.

## Setup

The examples live in the same Go module as the client, so no install step is needed.

```bash
export TWELVEDATA_API_KEY=<your-key>
```

## Run

```bash
go run ./examples/time_series
```
